# Documentation Images

## Setup Instructions

To display assignment screenshots in the main README:

1. Save the assignment screenshot images in this folder as:
   - `assignment-1.png` (Page 1 - Objective & Requirements)
   - `assignment-2.png` (Page 2 - Additional Requirements)
   - `assignment-3.png` (Page 3 - Technical Requirements)
   - `assignment-4.png` (Page 4 - Sample Requests)

2. Save example output screenshots as:
   - `example-1-output.png`
   - `example-2-output.png`
   - `example-3-output.png`
   - `example-4-output.png`
   - `example-5-output.png`

The README.md file already references these images.

## Alternative

If you don't have screenshots, the README still works - users can view JSON outputs in the `outputs/` folder.
